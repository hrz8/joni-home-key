# Joni Finding Home Key (Go 1.15)

## Quick Start

### Story

```json
This is what implemented in `main.go`

########
#......#
#.###..#
#...#.##
#o#....#
########

*details:
# is the wall
. is available route
o is person position
```

### Clue

```json
This is what implemented in `main.go`

1. Going north
2. Going East
3. Going South
```

### Usage

``` bash
$ go run main.go
```
*note: you can modify the input on `main.go` file either the `floor`, `person pos` and the `clue` as well.
